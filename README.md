# www.kotnik.si

## About the project
My personal web page build with Vue.js 3 and Bulma.  
Free to use.

## Update node on MacOS
```
brew update
```

## Update npm
```
npm install -g npm@latest
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
./build.sh
```
