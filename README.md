# www.kotnik.si

## About the project
My web page repository build with Vue.js 3 and Bulma.  
Free to use.


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
cp CNAME docs/
# or
./build.sh

```
