# www.kotnik.si

## About the project
My personal web page build with Vue.js 3 and Bulma.  
Free to use.

## Update npm
```
npm install -g npm@latest
```

## Project setup
```sh
npm install
```

### Compiles and hot-reloads for development
```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Deploy
```sh
npm run deploy
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
