# www.kotnik.si
My web page repository.  
Builded with Vue.js and Bulma.
> 

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run development
npm run hot

# build for production with minification
npm run production
```
