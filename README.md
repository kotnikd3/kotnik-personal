# My web page repository  

## www.kotnik.si  
Builded with Vue.js 2.x and Bulma. Free to use.

> 
## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run development
npm run hot

# build for production with minification
npm run production
```
